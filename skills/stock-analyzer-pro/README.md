# Stock Analyzer Pro 📊

专业的股票/基金分析技能 - 覆盖 A 股、港股、美股、基金的全维度分析

## 功能特性

### 📈 核心分析模块

1. **基础信息查询**
   - 实时行情（开高低收、成交量、涨跌幅）
   - 52 周高低点、市盈率、市净率
   - 市值、换手率等核心指标

2. **财务分析**
   - 三大财务报表分析
   - ROE、毛利率、净利率、负债率等
   - 财务健康度评分（0-100 分）
   - 成长性评估

3. **技术分析**
   - 均线系统（MA5/10/20/60/120/250）
   - MACD、KDJ、RSI、BOLL 指标
   - 趋势判断（多头/空头/震荡）
   - 支撑位/阻力位识别

4. **估值分析**
   - PE/PB 相对估值
   - 历史估值分位数
   - 合理价格区间
   - 安全边际计算

5. **风险分析**
   - 股价波动风险
   - 估值风险
   - 股东增减持
   - 限售股解禁
   - 股权质押

## 安装

### 1. 安装依赖

```bash
cd /home/admin/.openclaw/workspace/skills/stock-analyzer-pro
pip install -r requirements.txt
```

### 2. 测试安装

```bash
python scripts/main.py 600519
```

## 使用方法

### 命令行使用

```bash
# 分析 A 股
python scripts/main.py 600519
python scripts/main.py 000001.SZ

# 分析美股
python scripts/main.py AAPL
python scripts/main.py MSFT

# 分析基金
python scripts/main.py 000001
```

### 在 OpenClaw 中使用

安装技能后，直接向 AI 提问：

```
分析一下贵州茅台
看看 Apple 的股票情况
易方达蓝筹精选混合怎么样
分析 600519 的财务状况
```

## 注意事项

1. **网络连接**: 需要访问外网获取实时数据
2. **交易时间**: 非交易时间获取的是收盘价
3. **数据延迟**: 实时行情可能有 1-2 分钟延迟
4. **财务数据**: 财报发布后 1-2 天更新
