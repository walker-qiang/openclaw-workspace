## 🔧 技术架构

### 数据源优先级

```
1. 腾讯 API（主数据源）
   - A 股数据准确
   - 字段丰富
   
2. 新浪财经（备用）
   - 港股支持好
   - 数据稳定
   
3. 东方财富（财务数据）
   - 财务指标全
   - 需要 API Key（可选）
   
4. 雅虎财经（美股/港股）
   - 全球覆盖
   - 作为最后备选
```

### 数据验证流程

```
获取数据 → 格式验证 → 合理性检查 → 交叉验证 → 展示
    ↓          ↓           ↓           ↓
  失败→尝试下一个数据源
```
