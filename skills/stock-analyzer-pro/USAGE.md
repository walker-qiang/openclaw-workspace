# Stock Analyzer Pro - 使用指南

## 🎯 快速开始

### 在 OpenClaw 中使用

**无需记忆任何命令**，直接向 AI 提问即可：

```
分析一下贵州茅台
看看腾讯控股的股票
Apple 的财务状况怎么样
易方达蓝筹精选混合值得买吗
```

AI 会自动调用股票分析技能，生成专业分析报告。

## 📋 分析报告内容

每份分析报告包含：

### 1. 核心指标
- 当前价格、涨跌幅
- PE(TTM)、PB
- 市值
- 52 周范围

### 2. 财务健康度 (0-100 分)
- ROE（净资产收益率）
- 毛利率、净利率
- 负债率
- 成长性指标

### 3. 技术面分析
- 均线系统（MA5/10/20/60）
- MACD、KDJ、RSI 指标
- 支撑位/阻力位
- 趋势判断

### 4. 估值分析
- PE/PB 估值水平
- 合理价格区间
- 安全边际

### 5. 风险提示
- 波动风险
- 估值风险
- 流动性风险
- 其他风险因素

### 6. 综合评级
- 强烈推荐 / 推荐 / 中性 / 谨慎 / 回避
- 操作建议

## ⚠️ 注意事项

1. **数据延迟**: 实时行情可能有 1-2 分钟延迟
2. **非交易时间**: 非交易时段获取的是收盘价
3. **财务数据**: 财报发布后 1-2 天更新
4. **免责声明**: 报告仅供参考，不构成投资建议

## 🔧 故障排除

### 问题：无法获取行情数据

**原因**: 网络连接问题或 API 限流

**解决**: 
- 检查网络连接
- 稍后重试

### 问题：财务数据显示为 0

**原因**: 免费 API 提供的财务数据有限

**解决**: 
- 这是正常现象，核心指标（PE/PB）仍然准确
- 如需详细财务数据，可配置 Tushare API Key
